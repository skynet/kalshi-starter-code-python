Browse and reference all Material Design Icons available from Google Fonts. Click any icon to see its HTML code.

How to Use Material Design Icons

Step 1: Add the CDN link to your <head>

<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

Step 2: Use icons in your HTML

<span class="material-icons">wallet</span> <span class="material-icons">trending_up</span> <span class="material-icons">bar_chart</span>

Step 3: Style with CSS

.material-icons { color: #C5A028; /* Your gold color */ font-size: 24px; }
